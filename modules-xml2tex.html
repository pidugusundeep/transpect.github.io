xml2tex

https://github.com/transpect/xml2tex

Takes a XML and an xml2tex configuration file as input and converts the XML file according to the configuration to LaTeX.

Import URI

<p:import href="http://transpect.io/xml2tex/xpl/xml2tex.xpl"/>

Synopsis

<xml2tex:convert xmlns:xml2tex="http://transpect.io/xml2tex">
  <p:input port="source"/>                                <!-- a XML document-->
  <p:input port="conf"/>                                  <!-- xml2tex configuration, see example/conf-hubcssa.xml -->
  <p:output port="result"/>                               <!-- the LaTeX text file -->
  <p:with-option name="debug" select="'yes'"/>            <!-- "yes"|"no" -->
  <p:with-option name="debug-dir-uri" select="'debug'"/>  <!-- "debug" -->
  <p:with-option name="status-dir-uri" select="'status'"/><!-- "status" -->
</xml2tex:convert>
