hub2bits

https://github.com/transpect/hub2bits

This XProc step generates a ➼ BITS (Book Interchange Tag Suite) document from ➼ Hub XML.

Source ⬇

Import URI

<p:import href="http://transpect.io/hub2bits/xpl/hub2bits.xpl"/>

Synopsis

<tr:hub2bits xmlns:tr="http://transpect.io">
  <p:input port="source"/>                                <!-- Hub XML document -->
  <p:output port="result"/>                               <!-- Schematron report -->
  <p:with-option name="debug" select="'yes'"/>            <!-- "yes"|"no" -->
  <p:with-option name="debug-dir-uri" select="'debug'"/>  <!-- "debug" -->
  <p:with-option name="status-dir-uri" select="'status'"/><!-- "status" -->
</tr:hub2bits>
